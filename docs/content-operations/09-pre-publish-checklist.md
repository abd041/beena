# Pre-Publish Checklist

Complete before setting status to **Published** in CMS.

## Scientific & operational

- [ ] Claims are accurate and defensible for ophthalmic B2B audience
- [ ] Regulatory references (FDA, EMA, HTA) used correctly
- [ ] No confidential client information without approval
- [ ] Modality and indication context is specific (retina, gene therapy, etc.)
- [ ] Operational realities acknowledged (timelines, governance, evidence gaps)

## Editorial quality

- [ ] Tone is consultative — no hype or startup jargon
- [ ] **Publish readiness** panel shows no blocking errors
- [ ] ≥ 900 words and ≥ 4 `##` sections (articles)
- [ ] Executive summary and takeaways are substantive
- [ ] Second human review completed (required for publish tier)

## Structure & formatting

- [ ] One H1 on page (hero only); body uses `##` sections
- [ ] Paragraphs separated by blank lines
- [ ] No broken markdown or pasted HTML

## SEO

- [ ] Unique title and slug
- [ ] Excerpt 40–160 characters
- [ ] Meta description 120–160 characters (or strong excerpt fallback)
- [ ] Related service selected
- [ ] Cover image uploaded (1200×630 recommended)

## Internal linking

- [ ] Related service slug set
- [ ] Cross-links to portfolio/insights valid (manual check)
- [ ] CTA paths: `/book` primary, `/contact` secondary

## Trust & legal

- [ ] No guaranteed outcomes or comparative superiority without evidence
- [ ] Testimonials (if any) anonymized appropriately
- [ ] Disclaimers intact where required (metrics footnotes on site)

## Mobile & UX

- [ ] Preview live URL on mobile
- [ ] Cover image crops acceptably
- [ ] No horizontal overflow in prose

## Post-publish

- [ ] Verify `/insights/[slug]` returns 200
- [ ] Spot-check OG preview
- [ ] Confirm sitemap will include URL (automatic on publish)
